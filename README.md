# BlackRoad Deployment Package

Mass deployment of 100+ Cloudflare Pages projects with branded landing pages and API workers.

## Contents

- `template.html` - Dynamic landing page template with placeholders
- `deploy-all.sh` - Mass deployment script for 100 Pages projects
- `all-pages-projects.txt` - List of all 100 project names
- `blackroad-landing.html` - Main blackroad.io landing page
- `lucidia-landing.html` - Lucidia.earth landing page
- `workers/` - Cloudflare Workers (API, Auth, Analytics)

## Deployment Stats

- **100** Cloudflare Pages projects deployed
- **3** API Workers deployed
- **19** Custom domain CNAMEs configured
- **2** Full landing pages (BlackRoad + Lucidia)

## Workers

| Worker | URL | Purpose |
|--------|-----|---------|
| API | `blackroad-api-worker.workers.dev` | Central API gateway |
| Auth | `blackroad-auth-worker.workers.dev` | Authentication |
| Analytics | `blackroad-analytics-worker.workers.dev` | Event tracking |

## Usage

```bash
# Deploy all pages
bash deploy-all.sh

# Deploy individual worker
cd workers/api && wrangler deploy
```
